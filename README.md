## Installation
 via curl:
   `sh -c "$(curl -fsSL https://raw.githubusercontent.com/akabul0us/ohmyzsh_minimal/master/tools/install.sh)"`
 via wget:
   `sh -c "$(wget -qO- https://raw.githubusercontent.com/akabul0us/ohmyzsh_minimal/master/tools/install.sh)"`
 via fetch:
   `sh -c "$(fetch -o - https://raw.githubusercontent.com/akabul0us/ohmyzsh_minimal/master/tools/install.sh)"`

 As an alternative, you can first download the install script and run it afterwards:
   `wget https://raw.githubusercontent.com/akabul0us/ohmyzsh_minimal/master/tools/install.sh`
   `sh install.sh`
